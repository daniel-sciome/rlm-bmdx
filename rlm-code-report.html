pdf_text/3 files · 51 symbols
pdf_text is a from-scratch PDF parser library designed to extract semantic text from PDFs while replacing external PDF dependencies (requiring only zlib). It handles the complete PDF processing pipeline—resolving object graphs, decompressing streams, walking page trees, classifying text by semantic structure, and supporting advanced features like ToUnicode CMaps and character encodings. The package also includes a lossless PDF codec that can decompose PDFs into labeled byte spans for JSON storage and perfectly reconstruct them.
