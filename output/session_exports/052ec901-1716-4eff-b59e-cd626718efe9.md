### Data loading approach

1. **go_terms**: Parse `referenceUmapData.ts` with the same regex used in `go_gene_map.py` (reuse `parse_reference_umap()`)
2. **genes**: Built from `gene_consensus_merged.json` — consensus + moderate + single-mention counts. Also insert any gene from `go_term_genes.tsv` not in consensus (with evidence=NULL)
3. **papers**: Iterate all 7 `citegraph_output*/papers.json` files, dedup by paper_id (keep first seen, tag source)
4. **gene_go_terms**: Direct load from `go_term_genes.tsv`, skip rows with empty gene_symbol
5. **paper_genes**: From `gene_consensus_merged_extractions.json` (2,315 merged extractions) — flatten each extraction's `genes[]` array, normalize via `extract.normalize_gene()`
6. **paper_organs**: Same source, flatten `organs[]`, normalize via `extract.normalize_organ()`
7. **paper_claims**: Same source, flatten `claims[]`
8. **citation_edges**: Union all `citegraph_output*/edges.json` files, dedup
9. **pathways**: From `pathway_enrichment.tsv` — split KEGG rows and Reactome rows into unified format with `pathway_db` discriminator
